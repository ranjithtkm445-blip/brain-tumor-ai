# brain-tumor-ai
End-to-end brain tumor AI pipeline using BraTS 2020 MRI data. Performs tumor segmentation (2D U-Net), extracts 30 radiomics features, predicts 5 omics biomarkers (Random Forest, R2=0.91), and estimates survival risk (FusionNet, 93.5% accuracy) with GradCAM explainability and PDF reporting.
